# Celebrity Fashion Website - Spec-Driven Development (SDD)

## Project

**Project Name:** Celebrity Fashion

**Brand Name:** Celebrity Fashion

Build a modern, fast, responsive, SEO-optimized celebrity fashion news
and style website with a custom admin dashboard for publishing content.

-   NextAuth
